Unsupported Characters List

Version 1.20

Complete this form when you have checked both Y for IPL User Name/Comment Use AND N for Displays all IPL compatible fonts in the Nintendo DS Master ROM Submission Sheet. This form is used to indicate those characters that are not available in your game.

Notes:

  1. Because this page uses JavaScript, be sure to enable JavaScript.
  2. This application is designed to be used with Internet Explorer.

Updates

Version 1.20

  • The following ususable characters were found in the Character List for IPL menu, and have been deleted.
    0x02c6, 0x02dc, 0xff5b, 0xff5d, 0x0022, 0xff3e
  • The following characters had multiple check items in the Character List, which have been consolidated to one.
    0x00b1, 0x00d7, 0x00f7, 0x00b0, 0x2026, 0x201c, 0x201d

Version 1.10

  • European characters are now displayed on the confirmation screens.

Completing this form:

  1. Fill in all of the fields in Product & Contact Information below.
  2. Check all characters that are not supported in your game.
  3. If substitute characters are used, fill in the Substitute Characters field.
  4. Use the Comments field to provide any other information that needs to be noted.
  5. Click on the Create Submisson Document button to confirm that all unsupported characters have been checked.
  6. If the output information is correct, save the file.
  7. Submit the saved HTML file to Nintendo when submitting the Master ROM.
